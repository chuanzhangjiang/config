# 使用方式

配置命令

```
ln -s {源文件目录} {目标目录}
```
**注：两个目录都需要绝对路径**

| 配置文件或文件夹 | 位置               |
| ---------------- | ------------------ |
| neofetch         | ~/.config/neofetch |
| ranger           | ~/.config/ranger   |
| .bashrc          | ~/.bashrc          |
| .emacs           | ~/.emacs           |
| .xsession        | ~/.xsession        |
| .zshrc           | ~/.zshrc           |
